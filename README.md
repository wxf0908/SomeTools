# 🧰 SomeTools - 实用网页小工具集

一个基于纯 HTML/CSS/JavaScript 编写的在线工具集合，无需安装，打开即用，全部开源免费。

## ✨ 功能特性

- ⚡ **极速加载** - 纯静态页面，秒级加载，无需等待
- 🔒 **隐私保护** - 所有计算在本地完成，数据不上传服务器
- 📱 **响应式设计** - 完美适配手机、平板、电脑各种设备
- 🎨 **深色模式** - 支持明暗主题切换，保护视力
- ⌨️ **键盘快捷键** - 支持快速搜索和操作

## 🚀 快速开始

### 在线访问

直接打开 `index.html` 文件即可使用所有工具。

### 本地运行

```bash
# 克隆项目
git clone https://github.com/yourusername/SomeTools.git

# 进入项目目录
cd SomeTools

# 使用任意 HTTP 服务器启动（推荐）
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`
